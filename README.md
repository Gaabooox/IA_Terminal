# IA_Terminal

Implementando Ollama en kitty con zsh

## Descripcion

Con el còdigo puedes detectar errores que te muestre el terminal, enviando esa informaciòn a ollama y solicitandole de forma concisa explicando el problema, el porque y como poder solucionarlo.
AL detectar un error automaticamente se envia el error a la IA para su soluciòn, incluyendo funciones tambien para activar y desactivar la IA, sin generar SPAM.

## Implementaciòn

Agregar el còdigo al archivo de configuraciòn de la shell (.zshrc, .bashrc, etc.)
