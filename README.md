# IA_Terminal

Proyecto en curso actualmente solo funciona en Fedora, aun falta configurar la automatizaciòn.
Implementando Ollama en kitty con zsh

## Descripcion

Con el codigo puedes detectar errores que te muestre el terminal, enviando esa informaciòn a ollama y solicitandole de forma concisa explicando el problema, el porque y como poder solucionarlo.
AL detectar un error automaticamente se envia el error a la IA para su soluciòn, incluyendo funciones tambien para activar y desactivar la IA, sin generar SPAM.

## Implementacion

Agregar el còdigo al archivo de configuraciòn de la shell (.zshrc, .bashrc, etc.)
